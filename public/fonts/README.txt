PIXEL THEME FONT
================
Place your FreePixel.ttf file in this folder:

  public/fonts/FreePixel.ttf   (case-sensitive)

The app references it at the path /fonts/FreePixel.ttf (Vite serves /public at the web root).

If the file is missing, the app automatically falls back to Silkscreen,
which is embedded directly in src/styles/globals.css — so the pixel look
works either way. Drop in FreePixel.ttf to match the exact font from the mockup.
